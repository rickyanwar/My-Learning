    push() & pop()

    Fungsi push digunakan untuk menambahkan data elemen baru pada suatu array pada posisi index terakhir. Sebaliknya fungsi pop untuk menghapus elemen terakhir dari suatu array. Misalnya:
    vm.booksJSON.push({title: "buku baru"})

    unshift() & shift()

    fungsi unshift digunakan untuk menambahkan data element baru suatu array pada index posisi pertama(0). sebaliknya fungsi shift untuk menghapus element pertama dari array contoh :
    vm.booksJSON.shift()

    sort() & reverse()

    fungsi sort digunakan untuk mengurutkan data element pada array secara ascending, sedangkan reverse melakukan sebaliknya :
    vm.booksJSON.sort()

    splice()

    digunakan untuk menambahkan data element baru pada suatu array
    • parameter pertama merupakan posisi index dari data yang ditambahkan
    • parameter kedua jumlah element pada array yang akan di hapus
    contoh :
    vm.booksJSON.splice(3,0,'buku baru')
    contoh menghapus element dari suatu array :
    vm.booksJSON.splice(2,1) Perintah tersebut akan menghapus elemen array pada index ke-dua sebanyak satu elemen

    Fungsi set pada vue

    vue juga menyediakan fungsi built in untuk mengatasi kesalahan pada javascript ini, ika pada contohnya pada suatu element array dengan fungsi splice maka sebenarnya dapat dilakukan dengan fungsi vue :
    Vue.Set(data_array, index_yang_akan_diganti, nilai_baru)
    vm.$set(vm.booksJSON,1,{title: "Mastering hacking"})
    note: jika properti yang ingin ditambahkan lebih dari satu dapat menggunakan cara berikut vm.book = Object.assign({}, vm.book,{ title : 'ganti judul'})
